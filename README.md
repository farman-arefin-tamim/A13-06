# 🚀 DigiTools

## 📌 Project Description

DigiTools is a modern React-based web application where users can explore premium digital tools, add them to a cart, and manage their selections easily. The project focuses on clean UI design, smooth user experience, and dynamic state management.

---

## 🛠️ Technologies Used

* **React.js** – Component-based UI development
* **Tailwind CSS** – Utility-first styling
* **DaisyUI** – Pre-built UI components
* **JavaScript (ES6+)** – Logic and functionality
* **React Icons** – Icon integration

---

## ✨ Key Features

1. **Dynamic Product Listing**

   * Displays products using mapped data from JSON
   * Clean card-based UI with responsive design

2. **Add to Cart & Delete Functionality**

   * Users can add products to the cart
   * Remove items instantly with real-time UI update

3. **Live Cart Total Calculation**

   * Automatically calculates total price using `reduce()`
   * Updates instantly when items are added or removed

---

## 🎯 Extra Highlights

* Responsive design for mobile and desktop
* Tab-based navigation (Products ↔ Cart)
* Clean and modern UI with gradient styling
